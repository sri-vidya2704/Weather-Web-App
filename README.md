# Weather-Web-App
A responsive and user-friendly Weather Web App that provides real-time weather updates for any city. The app displays temperature, humidity and dynamic weather icons based on live weather data. Built using HTML, CSS, and JavaScript with API integration for accurate forecasts.

Features:
  Search weather by city name
  Displays temperature in Celsius
  Shows humidity levels
  Displays wind speed
  Dynamic weather condition icons
  Fully responsive design
  Fast and lightweight

Technologies Used:
  - HTML5
  - CSS3
  - JavaScript
  - Weather API
